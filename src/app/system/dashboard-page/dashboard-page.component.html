На графике представлен информационный срез по кластерам. Чтобы начать работу, выберите в первом поле интересующий вас кластер или оставьте "все", чтобы просмотреть информацию по всей строительной технике. Во втором поле выберите тип представления данных: среднее значение, сумма или медиана. В третьем поле выберите интересующий параметр для анализа. Далее вы автоматически получите график, упорядоченный по дате в соответствии с выбранными параметрами
